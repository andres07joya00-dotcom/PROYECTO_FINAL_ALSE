\doxysubsection{Detailed Description}
Punto de entrada principal de la aplicación de inventario. 

Este archivo inicializa el entorno Qt, establece la conexión con la base de datos mediante \doxylink{classDatabaseManager}{Database\+Manager} y crea la ventana principal de la aplicación.
