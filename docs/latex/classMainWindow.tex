\doxysubsection{Detailed Description}
Controlador principal de la aplicación de gestión de inventario. 

Esta clase orquesta la interacción entre el usuario y la base de datos. Sus responsabilidades incluyen\+:
\begin{DoxyItemize}
\item Inicializar la base de datos y la interfaz de usuario.
\item Gestionar el modelo de datos (QSql\+Query\+Model) y el filtrado (QSort\+Filter\+Proxy\+Model).
\item Manejar eventos de botones (CRUD, exportación, restauración).
\item Aplicar delegados personalizados para la visualización (\doxylink{classLowStockDelegate}{Low\+Stock\+Delegate}). 
\end{DoxyItemize}
